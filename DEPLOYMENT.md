# Final Deployment Guide

## Unity Submission Build

1. Open `MainPaintingScene`.
2. Assign trained ONNX models to the agent behaviour.
3. Verify that each style can be reproduced.
4. Build for Windows or macOS using Unity Build Settings.

## Report Export

Use the material in `Docs/acm_report_draft.md` as the basis for the ACM-format paper.

Recommended final report sections:

1. Abstract
2. Introduction
3. Related Work
4. Methodology
5. Technical Implementation
6. Experiments and Evaluation
7. Results
8. Discussion
9. Limitations and Future Work
10. Conclusion

## Final Submission Checklist

- Unity project opens without missing scripts.
- Demonstration collection process is shown.
- At least three trained models are exported as ONNX.
- Evaluation metrics are reported for all three styles.
- Visual trajectory comparisons are included.
- The video demo follows `Docs/video_demo_plan.md`.
